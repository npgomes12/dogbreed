<h1>Interactive Web Application with Stock Tracking, Dog Breed Explorer, and Voice Navigation</h1>

<h2>Description</h2>
This project consists of a multi-page web application built using HTML, CSS, and JavaScript. The application integrates multiple APIs to provide real-time stock market data, random dog images, dog breed information, and motivational quotes. Users can interact with the website through both traditional navigation and voice commands using speech recognition technology. The project demonstrates front-end web development, API integration, data visualization, and voice-controlled user interactions.
<br />

<h2>Languages and Utilities Used</h2>

- <b>HTML5</b>
- <b>CSS3</b>
- <b>JavaScript</b>
- <b>Chart.js</b>
- <b>Annyang Speech Recognition</b>
- <b>REST APIs</b>

<h2>APIs Used</h2>

- <b>Polygon.io Stock Market API</b>
- <b>Dog CEO API</b>
- <b>Dog API</b>
- <b>ZenQuotes API</b>
- <b>ApeWisdom API</b>

<h2>Environments Used</h2>

- <b>Visual Studio Code</b>
- <b>Windows 10/11</b>
- <b>Google Chrome</b>
